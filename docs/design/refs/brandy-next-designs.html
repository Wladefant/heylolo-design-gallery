P2 — the v1.1 viral loop (build starts ~Aug; design due end of July)

  1. Share-link preview + landing (N4 + founder ask) — the WhatsApp/iMessage share must look "super good": the kid's artwork/quiz-result AS the OG preview image + app link; plus the no-install playable landing page it opens. This is the single highest-leverage undesigned surface (it's the growth engine).

  4. Friday Drop moment — the weekly quiz-deck arrival: how a new deck announces itself (in-app moment + optional parent digest slot). Content is authored (13 decks); the arrival UX is undesigned.
