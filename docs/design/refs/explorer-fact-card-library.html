1. Card format

Every card is a fixed record so a card-hit renders identically to a live-model fallback (§2) — same shape, curated content:

  • Recognition key(s) — object noun(s) the vision classifier maps to this card (synonyms folded in, e.g. cup, mug).
  • Gasp + hook, ×2 variants — the Gasp device + a question back to the kid, so a repeat sighting doesn't feel canned. EN and DE each get their own two variants (DE is written to its own rhythm, not a translation, per the glossary's Du-voice + parataxis rules).
  • Whisper-secret fact — one T1-safe common-knowledge fact, delivered as a conspiratorial aside, with a T3 upgrade folded in after an em-dash for the 8–9 tier (a sharper detail on the SAME fact, never a different one, so a re-hit at an older age still feels continuous).
  • Shelf artifact — which Wonder Shelf collection this discovery mints into via the Explorer-capture earn rule (wonder-shelf-launch-catalog.md §2 row 5, "Explorer's Lens"). A strong thematic match (moon→Star Voyager, dog→Animal Friends) mints into that interest set instead of the generic First-Adventures slot — same event, richer payoff.
  • Share-clip rating — high/med/low. Best 15s demo clips are high-surprise-fact + visible Gasp + kid reaction; everyday objects (spoon, key) are low (skip the reel), high-wow objects (moon, snail, rainbow) are high.

All facts below are bulletproof common knowledge — nothing requiring a lookup, so nothing to get wrong on-device.

2. Fallback grammar

When Gemini Live recognizes an object outside the top-40, the response must still feel like a card. Prompt fragment for the Explorer system instruction (additive, sits alongside the persona-devices block in kids-voice-conversation-design.md §5a):

    WHEN YOU RECOGNIZE SOMETHING IN THE CAMERA:
    1. GASP first — a genuine, brief reaction of noticing ("Ooh!", "Whoa—"),
       before you name it. Never lead with the name.
    2. NAME it warmly, like you're delighted to meet it, not identifying a
       specimen ("A snail!" not "This appears to be a garden snail.").
    3. ONE wonder — a single true, simple, common-knowledge fact a curious
       kid would love. If unsure it's true, skip it — say what you noticed
       instead ("Look at that shiny shell!").
    4. QUESTION back — end on ONE question that hands the turn to the child
       (never two), inviting them to look closer, guess, or share.
    Keep the whole reaction to 1–2 short sentences. Never lecture. Never
    stack facts.

This is deliberately the same gasp → name → wonder → question skeleton as every curated card, so the two layers are indistinguishable in feel — only the curated layer is guaranteed word-perfect and pre-approved.

3. "LoLo sees it!" moment spec

The visual+haptic cue (#104) that stamps a recognition as a discovery, not just a reply. Maps onto the shipped feedback module (src/shared/feedback/index.ts) and the shelf's tier language (wonder-shelf-launch-catalog.md §2: whisper/moment/landmark):

Path Confidence Cue fires? Feedback call Why
Card-hit (one of the 40) n/a — curated match is binary Always feedback.success("S") + disc/avatar "notice" bloom Curated = guaranteed-good content; the cue is a promise the library keeps every time.
Card-hit → first-ever for this child n/a Always, upgraded feedback.success("M") + shelf artifact flies to Wonder Shelf First sighting mints the Explorer's Lens / themed artifact (catalog §2 row 5) — a moment, not a whisper.
Generic (fallback grammar) hit model confidence ≥ threshold (tune in QA; start ~0.7) Only above threshold feedback.success("S") Below threshold, LoLo may still answer (§2) but the badge is suppressed — an uncertain guess shouldn't wear the same badge as a sure one.
Generic hit, low confidence below threshold No cue none Prevents the cue becoming meaningless noise on every camera frame.

Rule of thumb: curated hits are cheap to trust (hand-checked at authoring time) so they always cue; generic hits are model guesses, so the cue is gated — keeping "LoLo sees it!" a signal the child learns to trust.
